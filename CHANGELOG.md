## [5.0.0] — 2026-06-15 🚀 Pro Max Edition

### Added
- Complete Flutter Web portfolio with cyberpunk neon aesthetic
- Riverpod 2.x state management (theme, language, filters)
- GoRouter navigation with hash-based section routing
- Animated particle field (canvas-based, 60 particles)
- Rotating gradient avatar ring with pulse glow
- Typing animation cycling through 8 developer roles
- Dark / Light mode toggle with persistent state
- Burmese / English language toggle (🇲🇲 / 🌐)
- Animated neon progress bars (22 skills)
- Tech chip cloud (38 technologies)
- Project section with featured cards + category filter
- 43 GitHub account grid with clickable links
- 16 App cards with live + code links
- 38+ Lovable PWA link grid with expand/collapse
- 21 email addresses with one-click copy
- Contact form with validation
- 10 social platform cards
- Scroll progress bar (neon gradient)
- Back-to-top button
- Cyberpunk grid background canvas
- Full CI/CD: ci.yml, build_web.yml, deploy_web.yml, build_android.yml, build_ios.yml
- Firebase Hosting config with security headers + cache rules
- PWA manifest with icons + screenshots
- SEO-optimized index.html with JSON-LD structured data + OG + Twitter cards
- Custom loading screen (animated MKA logo)

### Architecture
- Clean Architecture layer separation
- Feature-first folder structure
- Local data source pattern (PortfolioLocalSource)
- Riverpod providers for all state (theme, language, filters, data)
- Responsive helper (mobile / tablet / desktop breakpoints)
- Reusable widget library (GlowCard, NeonButton, SectionTitle, NeonProgressBar, CyberFrame)
