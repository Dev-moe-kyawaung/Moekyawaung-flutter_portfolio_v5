# Contributing

Thanks for your interest in contributing to this portfolio!

## Getting Started

```bash
git clone https://github.com/Dev-moe-kyawaung/Moekyawaung-flutter_portfolio_v5.git
cd Moekyawaung-flutter_portfolio_v5
flutter pub get
flutter run -d chrome
```

## Code Style

- Follow `analysis_options.yaml` rules
- Run `flutter analyze` before committing
- Use `const` constructors wherever possible
- Prefer `Riverpod` providers over local state for shared data

## Commit Convention

```
feat: add new section
fix: correct nav scroll offset
style: update neon glow intensity
refactor: extract widget to core/widgets
docs: update README
chore: bump dependencies
```

## Pull Requests

1. Fork the repository
2. Create a feature branch: `git checkout -b feat/your-feature`
3. Commit changes: `git commit -m 'feat: your feature'`
4. Push: `git push origin feat/your-feature`
5. Open a Pull Request
